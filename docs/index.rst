BPU-SDK Documentation
============================

BPU-SDK是由 `广东省智能科学与技术研究院 <https://gdiist.cn/>`_ 为自研类脑芯片开发的软件栈，
支持对 `Brainpy <https://github.com/brainpy/BrainPy>`_ 框架解析编译, 提供了丰富的Python语言API库,
支持常用的机器学习、深度学习、类脑计算算法及模型。

BPU-SDK 包含以下功能优势：
   * 集成类脑编译各个环节，减少编译时间，提高编译效率
   * 具备更简单的安装和配置过程，降低了用户学习曲线
   * 支持编译丰富的类脑算法，提供更好的兼容性和移植性
   * 充分利用硬件特性，支持超大规模类脑网络计算模拟
   * 集中管理编译相关的组件和逻辑，有利于后续的维护和升级


.. note::

   版本持续更新中
